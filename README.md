# Building-Damage-assessment-in-the-event-of-natural-disasters-using-satellite-images-

This project uses satellite imagery to automatically evaluate the impact of natural disasters like earthquakes, floods, or hurricanes. By comparing pre-disaster and post-disaster images, a deep learning model—typically a semantic segmentation network such as U-Net—identifies buildings and classifies their condition. The pipeline involves data preprocessing, augmentation, and training on labeled image–mask pairs, enabling rapid, large-scale damage mapping. This helps governments and humanitarian organizations prioritize rescue efforts, allocate resources efficiently, and accelerate disaster response.
